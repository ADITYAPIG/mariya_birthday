# Happy Birthday Mariya 💛

A 3-screen birthday website:
1. **Do You Love Me?** — YES/NO with a jumping, shrinking, vanishing NO button
2. **Happy Birthday reveal** — full-screen photo
3. **Our Memories** — a 9-photo timeline

## Files
```
index.html
css/style.css
js/script.js
images/  (her-photo.png + memory-01.png ... memory-09.png)
```

## Deploy to GitHub Pages
1. Create a new repository on GitHub (e.g. `happy-birthday-mariya`).
2. Upload these files/folders keeping the exact same structure (don't rename anything, the code links to these exact paths).
3. Go to the repo's **Settings → Pages**.
4. Under "Build and deployment", set **Source** to `Deploy from a branch`, branch `main`, folder `/ (root)`. Save.
5. Wait a minute or two — GitHub will give you a live URL like `https://yourusername.github.io/happy-birthday-mariya/`.

That's it — no build step, no dependencies to install.
